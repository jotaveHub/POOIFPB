# ☕ Programação Orientada a Objetos (POO)

Este repositório foi criado para centralizar os códigos desenvolvidos durante as aulas da disciplina de **Programação Orientada a Objetos**, além da resolução de desafios propostos.

O objetivo principal é a aplicação prática dos pilares do paradigma orientado a objetos, utilizando a linguagem **Java** para construir sistemas modulares, escaláveis e de fácil manutenção.

---

## 📌 Conceitos Explorados

Durante as aulas e nos exercícios resolvidos, foram aplicados os seguintes conceitos:

*   **Abstração:** Modelagem de entidades do mundo real em classes.
*   **Encapsulamento:** Proteção de dados utilizando modificadores de acesso e métodos acessores (*getters* e *setters*).
*   **Herança:** Reaproveitamento de código e criação de hierarquias entre classes.
*   **Polimorfismo:** Implementação de comportamentos específicos através de sobrescrita e sobrecarga de métodos.
*   **Interfaces e Classes Abstratas:** Definição de contratos e moldes para o sistema.
